# @forecasting-mrp/api

Backend API for ForecastingMRP built with NestJS (TypeScript).

## Tech Stack

- **Framework:** NestJS 10
- **ORM:** TypeORM
- **Database:** PostgreSQL 16 with TimescaleDB
- **Queue:** BullMQ + Redis
- **Auth:** JWT + Passport
- **Docs:** Swagger/OpenAPI

## Development

```bash
pnpm dev
```

Runs on `http://localhost:3001`.
