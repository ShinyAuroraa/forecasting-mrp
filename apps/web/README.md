# @forecasting-mrp/web

Frontend application for ForecastingMRP built with Next.js 14 (App Router), Tailwind CSS, Shadcn/UI, and Apache ECharts.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Shadcn/UI
- **Charts:** Apache ECharts
- **State:** Zustand + TanStack Query
- **Validation:** Zod

## Development

```bash
pnpm dev
```

Runs on `http://localhost:3000`.
